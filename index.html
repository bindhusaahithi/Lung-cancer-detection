Lung Cancer Detection using Deep Learning

A deep learning project that classifies lung CT scan images using Convolutional Neural Networks (CNNs). The project includes both binary and multi-class models for detecting lung abnormalities.

View GitHub

Project Overview

This project uses deep learning techniques to classify lung CT scan images. Two CNN models were built:

  • Binary classification: Cancer vs Normal
  • Multi-class classification: Benign, Malignant, Normal
