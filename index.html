DEEP LEARNING PROJECT

Lung Cancer Detection using Deep Learning

A deep learning project using Convolutional Neural Networks (CNNs) to classify lung CT scan images and detect possible lung abnormalities.

View GitHub View Results

Project Overview

This project uses deep learning techniques to classify lung CT scan images and identify patterns associated with lung cancer. Medical image classification is an important AI application because neural networks can detect subtle patterns in imaging data.

The project uses Convolutional Neural Networks (CNNs) to analyze CT scan images and classify them into multiple categories including Normal, Benign, and Malignant.

This project demonstrates key machine learning and deep learning skills including image preprocessing, model training, performance evaluation, visualization, and real-world prediction.

Tools & Technologies

  • Python
  • TensorFlow / Keras
  • OpenCV
  • NumPy
  • Matplotlib
  • Scikit-learn
  • Seaborn
  • Jupyter Notebook

Dataset

Dataset: IQ-OTHNCCD Lung Cancer Dataset

The dataset contains CT scan images categorized into three classes:

  • Normal
  • Benign
  • Malignant

Images were resized, normalized, and organized into folder-based class structures for training deep learning models.

Project Workflow

  1. Data Preparation – Organizing CT scan images into class folders
  2. Image Preprocessing – Resizing, normalization, and grayscale processing
  3. Train-Test Split – Dividing data into training and testing sets
  4. Model Design – Building CNN architectures
  5. Model Training – Training the neural network using TensorFlow
  6. Evaluation – Measuring accuracy, loss, and confusion matrix
  7. Prediction – Testing the trained model on unseen CT scan images
