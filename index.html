DEEP LEARNING PROJECT

Lung Cancer Detection using Deep Learning

A deep learning project using Convolutional Neural Networks (CNNs) to classify lung CT scan images. This project includes binary and multi-class models for detecting lung abnormalities.

View GitHub View Results

Project Overview

This project applies deep learning techniques to classify lung CT scan images and identify patterns related to lung abnormalities. Medical image classification is an important application of AI because neural networks can learn subtle visual features from imaging data.

Two versions of the project were developed. The first is a basic binary classification model to distinguish between Cancer and Normal CT scans. The second is an advanced multi-class model that classifies scans into Benign, Malignant, and Normal categories.

This project demonstrates practical skills in image preprocessing, CNN model design, training, evaluation, result visualization, and prediction generation.

Dataset

Dataset: IQ-OTHNCCD Lung Cancer Dataset

The dataset contains CT scan images categorized into three classes:

  • Normal
  • Benign
  • Malignant

Images were resized, normalized, and organized into class-based folders for model training.

Project Workflow

  1. Data Preparation – Organizing CT scan images into class folders
  2. Image Preprocessing – Resizing, normalization, and grayscale conversion
  3. Train-Test Split – Dividing data into training and testing sets
  4. Model Design – Building CNN architectures
  5. Model Training – Training the neural network using TensorFlow/Keras
  6. Evaluation – Measuring accuracy, loss, and confusion matrix
  7. Prediction – Testing the trained model on unseen CT scan images

Project Objectives

  • Build a CNN-based workflow for lung CT scan classification.
  • Compare binary and multi-class classification approaches.
  • Visualize model learning using accuracy and loss curves.
  • Evaluate class-level performance using confusion matrix heatmap.
  • Demonstrate real example predictions on unseen CT scan images.

Key Insights

  • The CNN model successfully learned meaningful visual patterns from lung CT scan images.
  • Training accuracy improved steadily while loss consistently decreased.
  • The confusion matrix helped validate class-level model performance.
  • The multi-class approach made the project more realistic and more representative of real medical classification tasks.
  • Prediction examples demonstrate how the trained model can classify unseen lung CT scan images.

Why This Project Matters

Lung cancer detection is an important use case in medical AI because imaging data contains rich patterns that deep learning models can analyze efficiently. This project demonstrates how CNNs can be applied to CT scan images for automated classification tasks.

Beyond the healthcare context, this project also reflects important machine learning engineering skills: data preparation, preprocessing, model training, evaluation, visualization, and presenting results in a structured portfolio format.
