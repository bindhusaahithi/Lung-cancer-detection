DEEP LEARNING PROJECT

Lung Cancer Detection using Deep Learning

A deep learning project using Convolutional Neural Networks (CNNs) to classify lung CT scan images and detect lung abnormalities through binary and multi-class image classification.

View GitHub View Results

Project Overview

This project uses deep learning techniques to classify lung CT scan images and identify patterns related to lung abnormalities. Medical image classification is an important AI application because convolutional neural networks can learn subtle visual features directly from imaging data.

Two versions of the project were developed. The first is a binary classification model for distinguishing between Cancer and Normal CT scans. The second is an advanced multi-class model for classifying CT scans into Benign, Malignant, and Normal categories.

This project demonstrates practical skills in image preprocessing, CNN model building, training, evaluation, result visualization, model saving, and prediction on unseen medical images.

Dataset

Dataset: IQ-OTHNCCD Lung Cancer Dataset

The dataset contains CT scan images categorized into the following classes:

  • Normal
  • Benign
  • Malignant

Images were resized, normalized, and organized into class-based folders for model training and testing.

Project Workflow

  1. Data preparation and folder organization of CT scan images
  2. Image preprocessing including resizing and normalization
  3. Train-test split for model development
  4. CNN architecture design using convolutional and dense layers
  5. Model training using TensorFlow / Keras
  6. Performance evaluation using accuracy, loss, and confusion matrix
  7. Prediction on unseen CT scan images
  8. Saving trained models for later reuse

Project Objectives

  • Build a CNN-based workflow for lung CT scan image classification
  • Compare binary and multi-class classification approaches
  • Visualize model learning through training accuracy and loss curves
  • Evaluate class-level performance using confusion matrix heatmap
  • Demonstrate real model predictions on unseen CT scan images

Why This Project Matters

Lung cancer detection is an important use case in healthcare AI because CT scan images contain rich visual patterns that deep learning models can analyze efficiently. This project shows how CNNs can support image-based classification tasks in a structured machine learning workflow.

Beyond the medical context, this project also demonstrates strong machine learning engineering skills including data handling, preprocessing, training, evaluation, result visualization, and project presentation.
