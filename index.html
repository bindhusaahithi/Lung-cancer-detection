Lung Cancer Detection using Deep Learning

A deep learning project using Convolutional Neural Networks (CNNs) to classify lung CT scan images. This project includes both binary and multi-class models for detecting Normal, Benign, and Malignant lung conditions.

View GitHub Repository View Results

Project Overview

This project focuses on medical image classification using deep learning. CT scan images were preprocessed and used to train CNN-based models for identifying cancer-related abnormalities in the lungs.

Two versions of the project were developed. The first is a basic binary classification model for distinguishing between Cancer and Normal cases. The second is an advanced multi-class model designed to classify CT scans into Benign, Malignant, and Normal categories.

This project demonstrates practical skills in computer vision, deep learning, medical image preprocessing, model evaluation, result interpretation, and project presentation.

Dataset

Dataset: IQ-OTHNCCD Lung Cancer Dataset

The dataset contains CT scan images grouped into medically relevant classes.

  • Normal
  • Benign
  • Malignant

Images were resized, normalized, and organized into folder-based class structures for model training and evaluation.

Project Workflow

  1. Data Preparation: Organized CT scan images into binary and multi-class folder structures.
  2. Preprocessing: Applied resizing, grayscale formatting, and normalization.
  3. Train-Test Split: Split the dataset into training and testing subsets.
  4. Model Building: Built CNN architectures using convolution, pooling, flattening, and dense layers.
  5. Training: Trained the models on CT scan images using TensorFlow/Keras.
  6. Evaluation: Measured model performance using training accuracy, loss curves, confusion matrix, and prediction outputs.
  7. Model Saving: Saved trained models in `.keras` format.
  8. Prediction: Loaded the trained models and classified unseen CT scan images.

Project Objectives

  • Build a CNN-based pipeline for lung CT scan classification.
  • Compare binary and multi-class deep learning approaches.
  • Visualize model learning using accuracy and loss curves.
  • Interpret class-level performance using confusion matrix heatmap.
  • Demonstrate real predictions on unseen CT scan images.
