Lung Cancer Detection using Deep Learning

Deep Learning project using CNN to classify lung CT scan images.

View GitHub

Project Overview

This project uses Convolutional Neural Networks (CNNs) to detect lung cancer from CT scan images. The model was trained to classify scans into multiple categories including Normal, Benign, and Malignant.

The goal of this project is to demonstrate deep learning techniques applied to medical imaging and build an automated system for lung cancer detection.

Tools & Technologies

  • Python
  • TensorFlow / Keras
  • NumPy
  • OpenCV
  • Matplotlib
  • Scikit-learn
  • Jupyter Notebook

Dataset

Dataset: IQ-OTHNCCD Lung Cancer Dataset

The dataset contains CT scan images categorized into three classes:

  • Normal
  • Benign
  • Malignant
