Project Overview

From CT Images To Model Predictions

Problem Framing

The project studies how convolutional neural networks can learn visual patterns from lung CT scan images and classify them into clinically meaningful categories in a controlled dataset experiment.

Dataset

IQ-OTH/NCCD Lung Cancer Dataset with normal, benign, and malignant CT images.

Dataset Link: Kaggle

Preprocessing

Images are converted to grayscale, resized to 128 x 128, normalized, and split for training/testing.

Modeling

Sequential CNN architecture with convolution, pooling, dense, dropout, and output layers.

Future Improvements

How This Could Be Extended

To move beyond a portfolio proof of concept, the next steps would be external validation, patient-level splitting, explainability methods, and deployment safeguards.

External Validation

Evaluate the model on CT images from a separate source to test generalization.

Explainability

Add Grad-CAM style heatmaps to show which scan regions influence predictions.

Deployment Safety

Design clear human-review boundaries before any real-world medical use.

Responsible AI Note

Proof Of Concept, Not Clinical Diagnosis

This project is designed for portfolio and educational demonstration. Results are based on a dataset-specific train-test split and do not include external validation, patient-level splitting, clinical testing, or deployment safeguards. The model should not be used for medical decision-making.
