Lung Cancer Detection using Deep Learning

A deep learning project using Convolutional Neural Networks (CNNs) to classify lung CT scan images. This project includes both binary and multi-class models for detecting Normal, Benign, and Malignant lung conditions.

View GitHub Repository View Results

Project Overview

This project applies deep learning techniques to detect lung abnormalities from CT scan images. Medical image classification is an important real-world use case of AI because early identification of malignant patterns can help support faster and more accurate diagnostic workflows.

The dataset consists of lung CT scan images categorized into clinically meaningful classes. These images were preprocessed and used to train CNN-based deep learning models capable of learning visual patterns associated with cancer-related abnormalities.

Two versions of the project were built:

  • Basic CNN Model: Binary classification between Cancer and Normal.
  • Advanced CNN Model: Multi-class classification between Benign, Malignant, and Normal.

This project demonstrates practical skills in computer vision, image preprocessing, model training, performance evaluation, result interpretation, and portfolio-level project presentation.

Dataset

Dataset: IQ-OTHNCCD Lung Cancer Dataset

The dataset contains CT scan images grouped into medically relevant classes.

  • Normal
  • Benign
  • Malignant

Images were resized to 128×128, normalized, and organized into folder-based class structures for training and evaluation.

Project Workflow

  1. Data Preparation: Organized CT scan images into binary and multi-class folder structures.
  2. Preprocessing: Applied grayscale conversion, resizing, and normalization.
  3. Train-Test Split: Divided the dataset into training and testing subsets.
  4. Model Design: Built CNN architectures using convolution, pooling, flattening, and dense layers.
  5. Training: Trained the CNN models using TensorFlow/Keras.
  6. Evaluation: Measured performance using accuracy curves, loss curves, confusion matrix, and predictions.
  7. Model Saving: Saved the trained models in `.keras` format for later reuse.
  8. Prediction: Loaded saved models and predicted classes for unseen CT scan images.

Project Objectives

  • Build a CNN-based medical image classification pipeline for lung CT scan images.
  • Compare binary and multi-class classification approaches.
  • Visualize training behavior using model accuracy and loss graphs.
  • Interpret classification performance using confusion matrix and example predictions.
  • Present the complete project in a clean, recruiter-friendly portfolio format.
